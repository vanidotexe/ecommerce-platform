A modern e-commerce web application developed as a group project for the .NET labs at UAIC FII. The front end is built with Angular and TypeScript, the back end is powered by a .NET REST API, and Python is utilized for additional data processing (e.g. price recommendations). 

**Features**
- **Product Catalog:** Browse, search, and filter a wide variety of products with ease.
- **Shopping Cart & Checkout:** Seamlessly manage cart items and securely process customer orders.
- **User Authentication:** Registration, login, and profile management for shoppers.
- **Order Management:** View order history, track statuses, and manage returns.
- **Admin Tools:** A dedicated dashboard to manage product inventory, categories, and user accounts.
- **Responsive Design:** Optimized shopping experience across both desktop and mobile devices.
